If

If is used when you want something to only happen if a condition is true

if(true) {
    System.out.println("true");
}

true

If-Else

If-else is used when there is only two cases you want to happen. If the condition is true, one t hing happens, else, the other case happens.

if(false) {
    System.out.println("true");
}

else {
    System.out.println("false");
}

false

IF-elseif-else

If-elseif-else is used when there are more than two cases with different conditions you want to run. For every new case, you add a new elseif to the code segment

if(1 > 2) {
    System.out.println("1+2=4");
}

else if(2 > 1) {
    System.out.println("2+3=5");
}

else {
    System.out.println("2+2=3");
}

2+3=5
